/** * 財務記帳簿後端腳本 * 支援支出、收入、旅行紀錄、固定支出與帳戶設定同步 */ function doGet() { const ss = SpreadsheetApp.getActiveSpreadsheet(); const data = { records: getSheetData(ss, "records"), tripRecords: getSheetData(ss, "tripRecords"), accounts: getSheetData(ss, "accounts"), fixedExpenses: getSheetData(ss, "fixedExpenses"), settings: getSettings(ss) }; return ContentService.createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const postData = JSON.parse(e.postData.contents); const action = postData.action; const data = postData.data; switch(action) { case 'addRecord': appendRow(ss, "records", [data.id, data.date, data.type, data.name, data.category, data.amount, data.accountName, data.paymentMethod]); break; case 'addTripRecord': appendRow(ss, "tripRecords", [data.id, data.date, data.tripName, data.category, data.amount, data.accountName]); break; case 'deleteRecord': deleteRowById(ss, data.type === 'trip' ? "tripRecords" : "records", data.id); break; case 'updateAccounts': updateFullSheet(ss, "accounts", [["name", "balance", "currency"]], data.map(a => [a.name, a.balance, a.currency])); break; case 'updateFixedExpenses': updateFullSheet(ss, "fixedExpenses", [["id", "name", "amount", "accountName"]], data.map(f => [f.id, f.name, f.amount, f.accountName])); break; case 'updateSettings': // 確保 settings 包含密碼欄位 updateFullSheet(ss, "settings", [["key", "value"]], [ ["salary", data.salary], ["savingsGoal", data.savingsGoal], ["usdRate", data.usdRate], ["password", data.password || "1234"] ]); break; } return ContentService.createTextOutput("Success").setMimeType(ContentService.MimeType.TEXT); } // --- 輔助函式 --- function getSheetData(ss, name) { const sheet = ss.getSheetByName(name) || initSheet(ss, name); const values = sheet.getDataRange().getValues(); if (values.length <= 1) return []; const headers = values[0]; return values.slice(1).map(row => { let obj = {}; headers.forEach((h, i) => obj[h] = row[i]); return obj; }); } function getSettings(ss) { const sheet = ss.getSheetByName("settings") || initSheet(ss, "settings", [["key", "value"]]); const values = sheet.getDataRange().getValues(); let settings = { salary: 45000, savingsGoal: 10000, usdRate: 32.5, password: "1234" }; values.slice(1).forEach(row => { if (row[0]) { // 判斷是否為數字 const val = parseFloat(row[1]); settings[row[0]] = isNaN(val) ? row[1] : val; } }); return settings; } function appendRow(ss, name, rowData) { const sheet = ss.getSheetByName(name) || initSheet(ss, name); sheet.appendRow(rowData); } function deleteRowById(ss, name, id) { const sheet = ss.getSheetByName(name); if (!sheet) return; const values = sheet.getDataRange().getValues(); for (let i = values.length - 1; i >= 1; i--) { if (values[i][0] == id) { sheet.deleteRow(i + 1); break; } } } function updateFullSheet(ss, name, headers, rows) { const sheet = ss.getSheetByName(name) || initSheet(ss, name, headers); sheet.clear(); sheet.getRange(1, 1, headers.length, headers[0].length).setValues(headers); if (rows.length > 0) { sheet.getRange(2, 1, rows.length, rows[0].length).setValues(rows); } } function initSheet(ss, name, defaultHeaders) { let sheet = ss.getSheetByName(name); if (!sheet) { sheet = ss.insertSheet(name); const headers = defaultHeaders || getHeadersForName(name); sheet.getRange(1, 1, 1, headers[0].length).setValues(headers); } return sheet; } function getHeadersForName(name) { switch(name) { case "records": return [["id", "date", "type", "name", "category", "amount", "accountName", "paymentMethod"]]; case "tripRecords": return [["id", "date", "tripName", "category", "amount", "accountName"]]; case "accounts": return [["name", "balance", "currency"]]; case "fixedExpenses": return [["id", "name", "amount", "accountName"]]; case "settings": return [["key", "value"]]; default: return [["data"]]; } }
